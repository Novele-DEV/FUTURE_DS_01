# FUTURE_DS_01

## Objective
Analyse business sales data to identify revenue trends, top selling products, high value categories, and regional performance
In this project I aim to practice and develop my business analytics, data cleaning, KPI analysis, and insight generation skills.

## Tools Used
    -Python
    -Pandas
    -Streamlit
    -Matplotlib

## Hot to run
1. Install dependencies:
    pip install -r requirements.txt

2. Clean data:
    python src/data_cleaning.py

3.Run dashboard:
    cd dashboard
    streamlit run app.py
